# UX e Acessibilidade: Desenvolvendo Interfaces Inclusivas

Projeto simples que demonstra práticas de design inclusivo e acessibilidade digital. O foco está em criar uma interface legível, fácil de navegar e amigável para pessoas idosas.

## Recursos implementados

- HTML semântico com landmarks e títulos claros
- Botões de ajuste de tamanho de fonte
- Contraste alto ativável
- Link de salto para o conteúdo principal
- Foco visível em controles interativos
- Estrutura responsiva para dispositivos móveis
- Texto explicativo e instruções de teclado

## Como usar

1. Abra `index.html` em um navegador.
2. Use o botão "Aumentar texto" e "Diminuir texto" para ajustar a leitura.
3. Ative o modo de contraste alto para melhorar a visibilidade.
4. Navegue usando `Tab` e `Enter` para testar a acessibilidade por teclado.
